# Visualización de Datos – UOC 2025  
**Tecnología, salario e inteligencia artificial: el perfil del desarrollador en 2025**

Este repositorio contiene el código fuente de la práctica de Visualización de Datos del Máster en Ciencia de Datos de la UOC. El proyecto analiza la encuesta **Stack Overflow Developer Survey 2025** mediante cuatro gráficos interactivos que exploran la relación entre:

- Diversidad tecnológica (número de herramientas) y salario.
- Confianza en la IA frente a la experiencia técnica.
- Salario nominal vs. poder adquisitivo real por país.
- Modalidad de trabajo (remoto, híbrido, presencial) y su impacto en la compensación.

El objetivo es aplicar principios de **storytelling con datos**, diseñar visualizaciones claras y extraer conclusiones relevantes para profesionales del sector tecnológico.

## Ejecución en local (Vite + React)

Sigue estos pasos para levantar el proyecto en tu máquina:

### 1. Clonar el repositorio

```bash
git clone <url-del-repositorio>
cd visualizacion-uoc-2025
```

### 2. Instalar dependencias

Asegúrate de tener Node.js 18+ y npm (o yarn).

```bash
npm install
```

### 3. Iniciar el servidor de desarrollo

```bash
npm run dev
```

Vite iniciará el servidor en http://localhost:5173 (por defecto).
Abre esa dirección en tu navegador.
